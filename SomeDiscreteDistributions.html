Common Probability Distributions

There are some simple probability distributions that come up frequently in real-world situations.

Bernoulli Trial or Bernoulli Experiment

A Bernoulli Trial is a probability experiment where there are two outcomes (we usually call them success or failure, but it could be heads or tails, zero or one, etc.) We usually call the probability of success \(p\) and probability of failure \(q.\) Because there are only two outcomes and the sum of all the probabilities must be 1, we must have \(q = 1 - p.\)

Binomial Distribution

The binomial distribution describes the probability of \(X\) success in \(n\) trials of a Bernoulli experiment. (Notice that we capitalize \(X\) because it is a random variable. When we want to talk about a specific value that \(X\) might turn out to be, we use the lower case \(x.\)) The value of \(X\) could be anywhere from 0 to \(n,\) or \(X \in \{0, 1, ..., n\}\)

The probability of a single outcome with exactly \(x\) successes in \(n\) trials is \(p^x q^{n-x},\) because if there are \(x\) successes, there must be \(n - x\) failures. But those \(x\) successes could occur in any of the \(n\) trials, so there are \(_{n}C_x\) sequences with \(x\) successes in \(n\) trials.

This gives us the formula \[P(X = x) = p(x) = {}_{n}C_x p^x q^{n-x} = {}_{n}C_x p^x (1-p)^{n-x} =\frac{n!}{(n-x)!x!} p^x q^{n-x}.\]

We use the shorthand notation \(X ~ b(n, p)\) to indicate that the random variable \(X\) has a binomial distribution with corresponding to \(n\) trials and probability of success \(p.\) For example, if \(X\) is the number of ones when you roll a fair die 15 times, then \(X ~ b(15, 1/6)\).

The expected value or mean for \(b(n, p)\) is \(E(X) = np.\) The variance for for \(b(n,p)\) is \(\sigma^2= np(1-p),\) so the standard deviation for \(b(n, p)\) is \(\sqrt{np(1-p)}.\)

Geometric Distribution

For a binomial experiment, we perform a Bernoulli trial a fixed number of times and count the number of successes. Suppose that, instead, we perform a Bernoulli trial until we get a success and count the number of trials to the first success. Because the number of trials is not fixed, this is a different distribution from the binomial. We call it a geometric distribution.

CAUTION: there are actually TWO ways to define a geometric distribution. One is as the number of trials to the first success (so that first success is counted) and the other is as the number of failures to the first success (so that only the failures are counted.) If you’re setting up the problem, be sure you think clearly about which you’ll count. If the setup is given to you, you’ll need figure out from the context what is being counted, the failures, or the total trials (which would be one more than the failures.)

Using \(p\) to represent the probability of success, the pdf for a geometric distribution is

\[P\left(X=x\right)=(1–p)^{x-1}p \] If we count the number trials (i.e. counting the success at the end), the the mean number of trials to the first success is \[\mu=E\left(x\right)=\frac{1}{p} \] and the variance is \[\sigma^2=\frac{1-p}{p^2}\] so that the standard deviation is \[ \sigma=\frac{\sqrt{1-p}}{p}\] If we only count failures, then the mean shifts down by one to \[\mu=E\left(x\right)=\frac{1}{p} - 1 = \frac{1-p}{p}.\] (because we don’t count that final success), but that doesn’t affect the variance or standard deviation, which stay the same.

Exponential Distribution

The Exponential Distribution is the continuous analog of the geometric distribution. Instead of counting the number of trials to the first successful event, we count the time to the first event.

\[p\left(x\right)=\lambda e^{-\lambda x}, x \ge 0\] \[E\left(X\right)=\frac{1}{\lambda}\] \[\mathrm{Standard\ Deviation}=\frac{1}{\lambda}\]

Poisson Distribution

The Poisson Distribution.
\[P\left(k \mathrm{ \ events\ in\ time\ period}\right)=P\left(X=k\right)=e^{-\frac{\text{events}}{\text{time}}\cdot \text{time period}}\frac{\left(\frac{\text{events}}{\text{time}}\cdot \text{time period} \right)^k}{k!},\] where \(k = 0, 1, 2, ...\)

Let \(\mu = -\frac{\text{events}}{\text{time}}\cdot \text{time period}\). In terms of \(\mu,\) the pdf of the Poisson Distribution can be written as \[P\left(X=k\right)=e^{-\mu}\frac{\mu^k}{k!},\] where \(k = 0, 1, 2, ...\) The expected value is \(E(X) = \mu\) and the variance is \(\sigma^2 = \mu,\) so the standard deviation is \(\sigma = \sqrt{\mu}\)

The distribution of waiting times between arrivals (the “inter-arrival” times) in a Poisson process is exponential with \(\lambda = \frac{\text{events}}{\text{time}}\) so that the expected value of the waiting times is \(\frac{1}{\lambda}\)
