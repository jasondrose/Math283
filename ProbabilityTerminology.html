Probability Terminology

Random Variable: A variable that takes its values by chance.

Probability Experiment or Trial: A procedure that can be infinitely repeated and has a well-defined set of possible outcomes.

Sample Space: the set of outcomes of a probability experiment.

Event: a subset of the sample space

Example: The experiment of tossing a coin. There are two outcomes, H and T, so the sample space is S = {H, T}. The event of getting a heads is A = {H}. The event of getting tails is B = {T}.

The other possible events are \(A \cap B = \{ \}\) (nothing happens when you toss the coin) and \(A \cup B = S\) (something happens.)

Example: The experiment of rolling a six-sided die. There are six outcomes, 1, 2, 3, 4, 5, 6, so \(S =\{1, 2, 3, 4, 5, 6\}.\) The event of rolling a one is \(\{1\}.\) The event of rolling an odd number is \(\{1, 3, 5\}.\)

Two experiments that are very similar may have different sample spaces depending on the details.

Example: Let Experiment 1 be tossing two coins and let Experiment 2 be tossing two coins and counting the number of heads.

The sample space of Experiment 1 is \(S_1 = \{(T, T), (H, T), (T, H), (H, H)\},\) while the sample space for Experiment 2 is \(S_2 = \{0, 1, 2\}.\) To calculate the probabilities of events in Experiment 2, it is helpful to think about the closely related Experiment 1. For example, if we assume the coin is fair, then the probability of each outcome in Experiment 1 should be \(\frac{1}{4}.\)

The event \(\{1\}\) in Experiment 2 corresponds to the event \(\{(H, T), (T, H)\}\) in Experiment 1, so \(P(\{1\}) = \frac{1}{2}.\) Here \(P(\{1\})\) means the probability of the event \(\{1\}\) occurring.

In general, when we have an event, \(E,\) then \(P(E)\) means the probability that \(E\) occurs.

Some Probability Facts:

\(P(\{ \}) = 0\) (the probability that nothing happens when the experiment is performed is 0.)

\(P(S) = 1\) (the probability that something in the sample space happens when the experiment is performed is 1. An event with probability 1 is certain to happen.)

If two events, \(A\) and \(B,\) don’t share any outcomes, i.e \(A \cap B = \{ \}\), then we say that events \(A\) and \(B\) are mutually exclusive. In other words, because they don’t share any outcomes, when one event occurs, the other cannot occur.

If two events, A and B, are mutually exclusive, then \(P(A \text{ or } B) = P(A \cup B) = P(A) + P(B)\)

Example: When you roll a six-sided die, the events of rolling a number less than three and rolling a number greater than 3 are \(A = \{1, 2\}\) and \(B = \{4, 5, 6\}\)

These events are mutually exclusive, so \(P(A or B) = P(A) + P(B).\)

If \(A\) is an event, then the complement of \(A,\) denoted by \(A^c\) or \(\bar{A},\) is the set of all outcomes in the sample space that are not in \(A,\) or, in set-subtraction notation, \(S \setminus A.\)

Events \(A\) and \(A^c\) are mutually exclusive, so \(1 = P(S) = P(A \text{ or } A^c) = P(A) + P(A^c).\) Solving the equation \(1 = P(A) + P(A^c)\) for \(P(A^c)\) gives \(P(A^c) = 1 - P(A).\) In words, the probability of the complement of an event is 1 minus the probability of the event.

Example: If \(P(A) = \frac{1}{3},\) then \(P(A^c) = 1 - \frac{1}{3} = \frac{2}{3}.\)

Example: The probability that two randomly selected people share the same birthday is about \(\frac{1}{365}\) (leap years make reality a little trickier, but about \(\frac{1}{365}\) is close.) So the probability that they don’t share the same birthday is \(1 -\frac{1}{365} = \frac{364}{365}.\)

Conditional Probability

Sometimes we’re interested in the probability that an event occurs given that another event has occurred.

The probability that event \(A\) occurs given that \(B\) has occurred is denoted by \[P(A | B) = \frac{P(A \cap B)}{P(B)}\]

Example: An urn contains two red balls and one blue ball. Find the probability that you draw a red given that you have already drawn a blue. Assume that you draw without replacement.

If you know that you’ve removed the blue, the probability that you’ll get a red is 1. So P(R|B) = 1.

Another way would be to compute it would be to use the definition: \(P(R | B) = P(\text{draw Blue and then draw Red})/P(\text{Blue}).\) But the probability that you draw blue on the first draw is \(\frac{1}{3}\) and once the blue is removed, the probability that you draw a red is 1 (because there are no blues), so \(P(\text{draw Blue and then draw R}) = \frac{1}{3}\cdot 1 = \frac{1}{3}\) so

\(P(R | B) = P(\text{draw Blue and then draw Red})/P(\text{Blue}) = \frac{1/3}{1/3} = 1.\)

Independence

We say two events are independent if \[P(A \cap B) = P(A)\cdot P(B)\].

Notice that this means that when \(A\) and \(B\) are independent, \[P(A |B) = \frac{P(A \cap B)}{P(B)} = \frac{P(A)\cdot P(B)}{P(B)} = P(A)\]

So when events are independent, whether or not \(B\) has occurred has effect on whether or not \(A\) occurs.

In a symmetric way, \[P(B |A) = \frac{P(B \cap A)}{P(A)} = = \frac{P(B)\cdot P(A)}{P(A)} = P(B)\]

Example: find the probability that you toss a heads given that you have just tossed a tails.

Assuming a fair coin and no funny business with the toss, the tosses should be independent, so \(P(\text{H this time}|\text{T last time}) = P(\text{H this time}) = \frac{1}{2}.\)
